# Books


## Features
Management for:
- Books

## Technologies
- ASP.NET Core 3.1
- AutoMapper

## Installation

Restore the dependecies in the folder `TestBooks.API` root

```bash
dotnet restore
```

## Run the project
Go to the project root folder
```bash
dotnet run -p TestBooks.API
```

App Running must be running in ports:
```bash
 localhost:5001 OR localhost:5000
```
